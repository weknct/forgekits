# Process Decomposition (Lite)

A free taste of the **Process Decomposition Framework** — an AI-native methodology that turns any fuzzy business process into an executable architecture of **Humans-in-the-Middle**, **AI Skills**, and **AI Agents**. This Lite plugin gets you from "here's how it works today" to a classified, error-compounded process map.

> You cannot automate what you have not decomposed. Lite gives you the first two phases: stand up a durable workspace, then break the process into atomic, classified events with the error-compounding math that tells you where validation checkpoints are mandatory.

## What's included (free)

- **`process-decomposer`** — the orchestrator agent (Lite edition). Coaches you through Bootstrap and Process Decomposition.
- **`phase-1-bootstrap`** — stands up the `{process-slug}/` workspace (context, todos, insights, registries, outputs) so the process keeps state across sessions.
- **`phase-3-process-decomposition`** — event-storms the process, classifies each event as 🧑 Human / ⚡ Skill / 🤖 Agent / 🧠 LLM, maps decision gates, and runs the error-compounding analysis.

Both skills bundle their real templates and checklists.

## How to use

Start with the **`process-decomposer`** agent: it runs `phase-1-bootstrap` to create the workspace, then `phase-3-process-decomposition` to produce the classified process map. You can also invoke either skill directly.
