# Changelog

All notable changes to the Client Attraction Engine (Lite) plugin are documented here.
This project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-06-21

### Added
- Initial public release of the free Lite tier.
- **Split Test Lab** skill (`ab-variant-generator`): generates 3-5 angle-varied
  A/B variants of any marketing asset, with tone and industry adaptation and a
  testing protocol for choosing the winner.
- **Pipeline Health Check** skill (`pipeline-health-check`): 5-stage funnel
  diagnostics, industry benchmarks, ROI / cost-per-client calculation, and
  prioritized bottleneck fixes.
- Plugin manifest, README, and Lite license.
