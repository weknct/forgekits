# Client Attraction Engine (Lite)

A free taste of the **Client Attraction Engine** — an AI-native marketing system for building automated client-attraction pipelines. Designed for small businesses, consultants, and service providers.

The Lite plugin ships the two highest-leverage AI capabilities from the full system, so you can sharpen your existing marketing right now — no setup, no cost.

## What's included (free)

| Skill | What it does |
|-------|--------------|
| **Split Test Lab** (`ab-variant-generator`) | Generate 3-5 angle-varied A/B variants of any headline, email subject line, ad, or CTA — with tone/industry adaptation and a concrete testing protocol to pick the winner. |
| **Pipeline Health Check** (`pipeline-health-check`) | Diagnose your funnel stage by stage, compare your numbers to industry benchmarks, calculate ROI and cost-per-client, and get prioritized fixes for the bottleneck. |

## Install

```
/plugin install client-attraction-engine-lite@forgekits
```

Once installed, the skills auto-invoke when relevant, or you can ask for them by name — e.g. *"Generate A/B variants for this headline using the Split Test Lab skill"* or *"Run a pipeline health check on these numbers."*
